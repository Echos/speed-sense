## 短い説明（最大 132 文字）

動画の無音区間を自動検出してスキップ。あらゆる動画サイトで動作するスピード拡張機能。

---

## 詳細説明

SpeedSense はリアルタイムで動画の音声を解析し、無音・間延び区間を自動で高速スキップします。余計な待ち時間を削り、学習や視聴をもっと効率よく。

**仕組み**

Web Audio API を使って再生中の動画の音声信号を解析します。設定したしきい値を下回る無音を検出すると、再生速度をなめらかに上昇させます。音声が再開すると通常速度にゆっくり戻ります。解析はすべてローカルで完結し、音声データは一切記録・送信されません。

**主な機能**

- 無音自動スキップ — 無音区間を検出して自動的に早送り
- スムーズな速度変化 — 400 ms の線形補間で急な速度変化を防ぐ
- あらゆるサイトで動作 — `<video>` 要素があるサイトであればどこでも
- サイトごとの速度記憶 — ドメイン単位で再生速度を自動保存
- リアルタイム波形オーバーレイ — スクローリング波形とスペクトラグラムを動画上に表示（ドラッグ移動可、動画枠内に固定）
- 節約時間カウンター — 累積節約時間をリアルタイム表示
- 細かいカスタマイズ — 通常速度・スキップ速度・検出しきい値・遅延・シーク秒数をすべて調整可能
- キーボードショートカット — 速度・切り替え・シーク・しきい値の操作をお好みのキーに割り当て

**設定値**

| 項目 | デフォルト | 範囲 |
|------|-----------|------|
| 通常速度 | 1.0× | 0.05〜16.0× |
| スキップ速度 | 3.0× | 1.5〜16.0× |
| 無音しきい値 | 1.5% | 0.5〜8% |
| 無音遅延 | 250 ms | 50〜800 ms |
| シーク秒数 | 10 秒 | 1〜60 秒 |

**権限について**

- `storage` — 設定をデバイス上にローカル保存するために使用
- `tabs` — ポップアップとアクティブなタブ間の通信、およびサポートページを開くために使用
- `host_permissions（<all_urls>）` — video 要素を持つ任意のページに無音検出スクリプトを注入するために必要。各ページのデータは読み取りも送信も行いません

**プライバシー**

SpeedSense は個人データを一切収集しません。すべての処理はお使いのブラウザ内で完結します。詳細なプライバシーポリシー: https://echos.github.io/speed-sense/
