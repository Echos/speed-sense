# SpeedSense

無音区間を自動検出して高速スキップする Chrome 拡張機能。
`<video>` 要素を持つ全サイトで動作します。

---

## 機能

| 機能 | 説明 |
|------|------|
| **SpeedSense** | 無音・無声区間を自動検出して高速スキップ |
| **速度調整** | 通常速度・無音時速度を個別に設定 |
| **波形オーバーレイ** | 動画上にスクローリング波形＋スペクトラグラムをリアルタイム表示（ドラッグ移動可、動画枠内に固定） |
| **シーク** | 任意の秒数で巻き戻し／早送り |
| **キーボードショートカット** | 全操作をカスタムキーに割り当て可能 |
| **節約時間表示** | 累積した節約時間をリアルタイムで表示 |
| **設定永続化** | すべての設定を chrome.storage.local に保存 |

---

## インストール

1. `chrome://extensions/` を開く
2. **「デベロッパーモード」** を ON にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. このフォルダ（`youtube-smart-speed/`）を選択

---

## 設定項目

### 速度設定

| 項目 | デフォルト | 範囲 | 説明 |
|------|-----------|------|------|
| 通常速度 | 1.0x | 0.05〜16.0x | 音声あり時の再生速度 |
| スキップ速度 | 3.0x | 1.5〜16.0x | 無音と判定された時の速度 |
| シーク秒数 | 10秒 | 1〜60秒 | 巻き戻し/早送りの移動量 |

### オーバーレイ表示

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| スペクトラグラム | OFF | ONにすると波形下に周波数スペクトラグラムを追加表示 |

### 無音検出

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| しきい値 | 1.5% | 小さくすると小さな音でも「音声あり」と判定。ノイズが多い動画は上げる |
| 遅延 | 250ms | 無音と判定するまでの待機時間。短すぎると誤検知が増える |

### キーボードショートカット（デフォルト）

| アクション | デフォルトキー |
|-----------|--------------|
| 通常速度 +0.05x | Alt+↑ |
| 通常速度 −0.05x | Alt+↓ |
| SpeedSense ON/OFF | Alt+S |
| しきい値 +0.5% | Alt+→ |
| しきい値 −0.5% | Alt+← |
| 1.0x リセット トグル | Alt+R |
| 巻き戻し | 未設定（ポップアップで登録） |
| 早送り | 未設定（ポップアップで登録） |

## 仕組み

```
<video> 要素
  ↓
captureStream() で音声ストリームをキャプチャ（CORS 回避）
  ↓
Web Audio API (AnalyserNode, fftSize=256)
  ├─ getByteFrequencyData()  → 音量平均 → 無音判定 → setTargetSpeed() → 400ms 線形遷移
  └─ getByteTimeDomainData() → 波形 min/max エンベロープ → スクローリング Canvas（上段）
                              getByteFrequencyData() データ → スペクトラグラム → Canvas（下段）
  ↓
節約時間 = Σ (silenceSpeed / normalSpeed - 1) × 無音区間の実時間
```

---

## 注意事項

- **AudioContext**: ブラウザの Autoplay Policy により、最初のユーザー操作後に有効化されます
- **YouTube SPA**: URL 変更を MutationObserver で監視して自動再初期化します
- **字幕・BGM**: BGM が流れている講義動画などは無音と判定されない場合があります
